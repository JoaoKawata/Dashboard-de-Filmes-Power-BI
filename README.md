# 🎬 Dashboard de Filmes — Power BI

Este projeto apresenta um dashboard desenvolvido no Power BI com uma base de dados de filmes. O objetivo foi praticar análise de dados, criação de gráficos, uso de filtros e organização visual de um painel interativo.

## Prévia do Dashboard

![Prévia do Dashboard](images/dashboard-preview.png)

## Objetivo do Projeto

O objetivo deste projeto foi analisar uma base de filmes, observando informações como notas, gêneros, décadas, diretores e idioma dos filmes.

Com o dashboard, é possível visualizar:

- Total de filmes analisados
- Média das avaliações IMDb
- Média das avaliações dos críticos
- Média das avaliações do público
- Média da nota geral
- Top 10 filmes por nota geral
- Top 10 diretores por quantidade de filmes
- Quantidade de filmes por gênero
- Quantidade de filmes por década

## Base de Dados

A base utilizada foi obtida no Kaggle e contém informações sobre filmes, incluindo título, ano, gênero, diretor, idioma e diferentes tipos de avaliação.

## Recursos do Dashboard

O dashboard contém:

- Cards com indicadores principais
- Gráficos de barras
- Gráfico de rosca
- Filtros interativos por gênero, década e idioma
- Layout com tema visual inspirado em cinema, utilizando azul escuro e detalhes em dourado

## Ferramentas Utilizadas

- Power BI 
- Kaggle
